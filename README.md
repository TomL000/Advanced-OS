# Advanced Operating Systems Assignment for 3rd Year.

## This reposistory contains Tasks 1-3 all in bash that can be easially executed using "bash task().sh" within a linux subsystem like WSL to execute the file.

### Task 1 - Menu Options
1. Show the CPU & Memory Usage
2. Show the Top Processes
3. Kill a Process (Please don't kill vital processes like systemmd or sshd).
4. Disk Usage Inspection
5. Archival Logs
6. Exit

### Task 2 - Menu Options
1. View Pending Jobs
2. Submit a Job
3. Process Queue: Round Robin
4. Process Queue: Priority Scheduling
5. View Completed Jobs
6. Exit

### Task 3 - Menu Options
1. Submit Assignment
2. Check for Duplicates
3. List of Submissions
4. Login Attempts Log
5. Exit
